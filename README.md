# TCP Client-Server

## Descrizione
Questo progetto realizza una comunicazione client-server utilizzando il protocollo TCP.  
Il client invia un messaggio al server e riceve una risposta.

## Funzionamento
Il server si mette in ascolto su una porta.  
Il client si connette al server e invia un messaggio.  
Il server riceve il messaggio e risponde.  
Il client riceve la risposta e la mostra a schermo.

## Tecnologie
Java  
Socket TCP

## Esecuzione
Avviare prima il server.  
Avviare poi il client.  
Scrivere un messaggio da tastiera.

## Note
TCP è un protocollo affidabile.  
Garantisce la consegna e l’ordine dei dati.
